# Student Mental Health & Academic Pressure Analysis (2025)

## Project Overview
This project analyzes student mental health and academic pressure data
collected via a survey (Kaggle dataset: *Student Mental Health & Academic Pressure 2025*).

The goal of this analysis is to explore:
- Levels of academic pressure among students
- Frequency of study-related stress
- The relationship between academic pressure and stress
- Sleep duration patterns across pressure levels

The analysis was performed using Python in a Jupyter Notebook environment.
